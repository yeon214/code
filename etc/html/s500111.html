객체지향프로그래밍

  • 교과목해설집

    • 객체지향 프로그래밍의 근본적인 개념들인 클래스, 멤버변수, 멤버함수, 데이터 추상화, 인캡슐레이션, 상속성, 동적바인딩 등을 상세히 학습한다.

  • 수업계획서
    1. 객체지향 방식으로 작성된 클래스 프로그램을 분석할 수 있다.
    2. 주어진 간단한 문제에 대한 클래스를 설계하여 객체지향 방식으로 프로그램을 작성할 수 있다.
    3. 표준 템플릿 라이브러리(STL)을 사용하여 프로그램을 작성할 수 있다.
  • 개인적 의견

    • 1학년 때 배웠었던 C언어에 객체의 개념을 더한 과목을 배우는 것이 객체지향프로그래밍이라고 생각한다.
    객체는 C++, Python 등등 여러 언어에서 쓰이기 때문에 잘 알아둬야 한다고 생각한다.

  • 이 과목을 선정한 이유

    • 객체라는 개념이 중요하다고 생각하기도 하고, 지금 흥미있는 과목들 중 하나이기 때문이다.
